# Quantum CNN on MNIST

This is a small course experiment comparing a conventional convolutional neural network with a quantum convolution layer on the MNIST handwritten-digit dataset.

I built it while exploring TensorFlow, model evaluation, quantum circuits with Cirq, and TensorFlow Quantum. The work is kept as a learning snapshot rather than a maintained package.

## Run the notebook

Open `qcnn-mnist.ipynb` in Jupyter or Google Colab and run the cells in order. It uses Python 3.9 with TensorFlow, TensorFlow Quantum, Cirq, SymPy, NumPy, Matplotlib, pydot, and Graphviz.

TensorFlow Quantum has strict Python and TensorFlow compatibility requirements, so a modern environment may need older compatible package versions. The notebook may require small adjustments as those libraries change.

No hosted demo is available.
